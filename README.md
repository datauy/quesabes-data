
Liberando los datos de "que sabes", web de pedidos de informacion al estado uruguayo basada en software Alaveteli.

Scripts para consultar la base de datos de un sitio web de alaveteli, generar formatos apropiados para publicacion. 


== Datos generados

* Estadisticas
  	por cada organismo tenemos
  		organismo, cantidad_pedidos, estado  

  		donde cantidad de pedidos por estado (rechazado, esperando,
aceptado). El estado es puesto por el solicitante al clasificar el pedido. 

* Pedidos
        titulo, descripcion, fecha de realizado, estado

* Instituciones
    	nombre, sigla, email, website
